Personal Jekyll Blog

## Development Setup
* `rbenv install 3.1.0`
* `gem install bundler`
* `git clone git@github.com:dcangulo/davidangulo.xyz.git davidangulo`
* `cd davidangulo`
* `bundle install`
* `bundle exec jekyll serve`
